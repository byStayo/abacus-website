Financial Dashboard Kiosk

A low-cost, always-on market data display system built on Raspberry Pi. Designed to bring real-time financial information to physical spaces - office lobbies, trading floors, and home offices - without the complexity or cost of traditional digital signage solutions.

The Challenge

Our client wanted to display live portfolio metrics and market data in their office lobby. Traditional digital signage solutions quoted $2,000+ for hardware alone, required proprietary software subscriptions, and couldn't be easily customized to show the specific financial data they needed.

They needed something that was:

  • Affordable enough to deploy in multiple locations
  • Reliable for 24/7 unattended operation
  • Fully customizable to display their specific metrics
  • Energy efficient and low maintenance

Our Solution

We built a complete solution using a Raspberry Pi 4, a 7-inch touchscreen display, and a custom web application. The system boots directly into a full-screen dashboard that displays:

  • Real-time portfolio value with historical charts
  • Major market indices (S&P 500, NASDAQ, DOW)
  • Custom watchlist with price alerts
  • News ticker with financial headlines
  • Company-specific metrics and KPIs

The dashboard is built with the same design language as our Abacus Games - dark theme, clean typography, and intuitive data visualization. It automatically reconnects if the network drops and gracefully handles API rate limits.

Technical Implementation

The solution runs entirely on the Pi with minimal external dependencies:

  • Raspberry Pi OS Lite with Chromium in kiosk mode
  • Node.js backend for API aggregation and caching
  • React frontend with WebSocket updates
  • Automatic startup on boot with systemd
  • Remote management via SSH and VNC
  • Watchdog timer for automatic recovery from crashes
Raspberry Pi 4 Node.js React WebSocket Chromium Kiosk systemd

The Result

The first kiosk has been running continuously for 8+ months without a single manual restart. The client has since deployed three additional units across their offices. Total cost savings compared to commercial signage solutions: over $7,000 in the first year alone.

More importantly, the system serves as a constant, visible reminder of their financial goals - exactly the kind of "accessible financial literacy" that's at the heart of everything Abacus builds.
